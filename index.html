Internal Tool

Employee Management System

Packly — Order Management · Built & maintained, Nov 2025–Present

The problem

Order management ran on manual tracking — agents logging tasks by hand, no shared view of who was working what, and no way to see SLA risk until a ticket was already late. A 20+ person team had no dashboard to see team-wide load or performance at a glance.

What I built

  • A Google Apps Script + Google Sheets web app for agent task logging and shift time tracking.
  • A manager dashboard with color-coded status chips for at-a-glance team visibility.
  • Task categories matching real order-management workflows: New Order, Customer/Merchant No Response, Confirm But Not Shipped, Courier Follow-Up, and Problem Solving.
  • Live session + completion-rate tracking so a manager can see daily throughput without asking anyone.

Result

Still in active use by the team. Replaced manual sheet-juggling with a single source of truth — agents log in, see their own task cards; the manager opens one dashboard and sees total assigned, total completed, completion rate, and active agents in real time.

WooCommerce

Surgiicare Bangladesh

Web Management — Freelance · Jan 2025–Apr 2025

The problem

Two separate problems on one account: the client needed a new WooCommerce medical eCommerce store built from scratch, and their existing agency site had been touched by 100+ previous developers — slow, bloated, and running on redundant plugins.

What I did

  • Built a WooCommerce medical eCommerce store from a premium Elementor UI kit, customizing product, category, and checkout pages to match the client's branding.
  • Audited every plugin and media asset on the existing agency site to find what wasn't pulling weight.
  • Removed redundant plugins and rebuilt the setup around a lean Elementor Pro stack with caching.

Result

Cut load times and made the site noticeably easier to maintain going forward — both the new eStore and the rebuilt agency site shipped live under the Surgiicare brand.
